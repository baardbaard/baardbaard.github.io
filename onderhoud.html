Wordpress Onderhoud

WordPress is geweldige software om zelf je website mee bij te houden, maar je doet er verstandig aan om regelmatig updates te installeren. Doe je dit niet dan loop je het risico dat je site gehackt wordt, geïnfecteerd wordt met virussen of mallware of gewoon niet functioneert in nieuwe browsers.

Heb je de tijd of kennis niet om dit zelf te doen en host je je website bij mij, dan help ik je hier graag bij. Voor €30 per maand doe ik het volgende:

  • Maandelijkse update van WordPress en de plug-ins
  • Maandelijkse controle op virussen en mallware
  • Spambestrijding
  • Correctie van zichtbare fouten
  • Wekelijkse backup van je complete website inclusief database
  • Maandelijkse rapportage van eventuele problemen en oplossingen

Een compleet pakket waarmee je er zeker van kunt zijn dat je website optimaal functioneert en blijft functioneren.

Het eerste half jaar is het WordPress onderhoudspakket gratis voor alle nieuwe WordPress websites die gehost worden bij Baardbaard.
